The link to Quizzical.

A pop quiz game to examine the playerâ€™s extensive knowledge. Each round consists of five questions spanning a range of topics from science to literature.

Step 1

Upon clicking the "Start Quiz" button, you will encounter five single-choice questions, with each question having four corresponding answer options.

Step 2

You are allowed to select only one answer for each question. Once an answer is chosen, its background color will change to blue.

Step 3

After responding to all the questions and clicking the "Submit Answers" button, the background of the correct answers will turn green. On the other hand, the chosen incorrect answers will have a red background.

Step 4

At the bottom of the App, there will be your score, and a "Play Again" button.

This project involves building a quiz app using React with Emotion for styling.

Here's a concise explanation of the techniques and logic used in each file:

App.js: Defines styled components for layout and blob shapes. Manages the app's state, including trivia questions, current page, and scores. Fetches trivia data from an API using Axios. Renders different pages (StartPage and QuizPage) based on the current page state.

StartPage.js: Uses styled components to create a centered start page layout. Displays a title, description, and start button. Calls the handleCurrentPageChange function to switch to the quiz page when the button is clicked.

QuizPage.js: Uses styled components to create the quiz page layout. Manages state for selected answers and submission status. Maps over trivia data to generate individual Quiz components. Provides context for whether answers have been submitted. Handles answer selection, submission, and display of results and scores.

Quiz.js: Uses styled components to create a question and answer layout. Receives question and answer data as props. Displays the question and maps over answers to create Answer components. Checks and highlights selected and correct answers during different stages.

Answer.js: Uses styled components to create answer buttons. Receives answer data as props. Handles click events for selecting answers. Adjusts button styles based on selected, correct, and results display states.

In summary, the project uses React to create a quiz app with different components for the start page, quiz page, questions, and answers. Emotion is used for styling, and Axios is used for API data fetching. The app handles user interactions, displays questions and answers, tracks selected answers, and displays scores and results.
