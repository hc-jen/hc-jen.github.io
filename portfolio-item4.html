The title : -Movie Fight-

Which is the best movie!

This is still a work in progress. The link will be provided once the project is finished.

There will be two input search bar for the user to compare two movies in various aspects.

Step 1

The user has the option to input keywords in either of the search bars. Subsequently, a menu displaying suggested movies will appear based on the provided keywords.

Step 2

When the user clicks on one of the options from the menu, the movie's details will be showcased. These details encompass information such as the genre, North America box office earnings, and rating.

Step 3

Alternatively, if the user clicks outside of the menu, the menu will be closed.

Step 4

Once the user selects two movies, the "Run Comparison" button will become visible.

Step 5

Upon clicking the "Run Comparison" button, a winner board will be displayed, revealing which movie emerges victorious in each category.
