The title : -Movie Fight-

Which is the best movie!

The link to Movie Fight.

There will be two input search bars for the user to compare two movies in various aspects.

Step 1

The user has the option to input keywords in either of the search bars. Subsequently, a menu displaying suggested movies will appear based on the provided keywords.

Step 2

When the user clicks on one of the options from the menu, the movie's details will be showcased. These details encompass information such as the genre, North America box office earnings, and ratings.

Step 3

Alternatively, if the user clicks outside of the menu before chosing a movie, the menu will be closed.

Step 4

Once the user selects two movies, the App will autimatically show which movie wins in each category.

This project involves building a Movies Comparison App using React with regular CSS for styling.

Here's a concise explanation of the techniques and logic used in each file:

Upon clicking the "Run Comparison" button, a winner board will be displayed, revealing which movie emerges victorious in each category.

App.js: Manages the state of selected movies, tutorial visibility, and trigger for showing summaries. Renders the application header, autocomplete search areas, movie summaries, and a tutorial section based on state values.

Autocomplete.js: Detects clicks outside of the component to close the dropdown menu. Handles input changes, fetches movie data, and displays search results in a dropdown menu. Passes selected movie data to parent component when a movie is chosen.

MovieOption.js: Represents an individual movie option within the autocomplete dropdown. Displays the movie's title, year, and poster image. Handles clicks on the movie option, fetching more details using the OMDB API, and passing the selected movie data back to the parent component.

Summary.js: Displays a summary of individual movie details. Renders movie information, awards, box office, metascore, IMDb rating, and IMDb votes.

NewSummary.js: Displays a comparison summary between the two selected movies. Displays movie information, including title, year, genre, and plot, along with relevant statistics such as awards, box office, metascore, IMDb rating, and IMDb votes. Applies conditional styling to highlight the winner in each statistical category based on comparison results.

fetchData.js: Contains a utility function that fetches movie data from the OMDB API based on a search term. Uses axios to make the API request and returns an array of movie results or an empty array if no results are found.

Overall, this code creates a React application for comparing movie details and displaying summary information in a user-friendly interface. The components interact to fetch movie data, display search results, and visualize comparison summaries.
