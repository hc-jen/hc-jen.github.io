The title : -Tenzies-

Classic Dice Game!

Stacks: React.js, JavaScript, CSS, RWD

The link to Tenzies.

"Tenzies" is adapted from "Tenzi", a popular dice game that involves rolling and matching dice to achieve a certain pattern or combination.

Step 1

When the player selects a specific number and clicks on it, the corresponding dice associated with that number will change its color to green.

Step 2

After the player clicks the "Roll" button, all the dice except the green one will shuffle and display new numbers.

Step 3

Continue selecting the number you initially chose and then proceed to shuffle the dice in order to obtain another number of your choice.

Step 4

When the player successfully gathers all the dice displaying the chosen number, a confetti effect will be triggered to celebrate the victory. Additionally, the "Roll" button will be transformed into a "New Game" button.

When the player's count number is lower the Best Score, it will be the new Best Score, which is stored in the local storage.

Here are some functions and logic behind the app:

State Management using React Hooks: The useState hook is used to manage the state of various variables within the application. presentGame, dice, tenzies, and bestScore are all managed using this technique. The state allows for dynamic changes in the UI based on user interactions.

Effect Hook: The useEffect hook is employed to monitor changes in the dice state. When all dice are held and have the same value, the game detects a win condition ("Tenzies"). This effect triggers when the dice state changes.

Generating New Dice: The generateNewDie function generates a new dice object with a random value between 1 and 6 and a unique ID using the nanoid library. The allNewDice function populates the dice state with an initial set of 10 dice.

Holding Dice: The generateNewDie function generates a new dice object with a random value between 1 and 6 and a unique ID using the nanoid library. The allNewDice function populates the dice state with an initial set of 10 dice.

Generating New Dice: The holdDice function toggles the isHeld property of a specific die, which allows the user to freeze the value of that die between rolls.

Handling Game Over: The handleGameOver function updates the bestScore state and stores it in local storage if the player's score is better than the previous best score.
