The title : -Realtime Weather App-

A weather App covering across Taiwan.

The link to Realtime Weather App.

Fetching real-time weather data from Taiwan Open Weather Data platform to build a weather App covering all the cities and counties in Taiwan. The switch between the dark and light themes determined by the time of sunrise and sunset

Step 1

Upon clicking the cog image located in the top right corner, a pop-up list of cities and counties in Taiwan will appear, allowing the user to make selections.

Here are some functions and logic behind the app:

State Management with React Hooks: The code uses various React hooks like useState and useEffect to manage and update the state of components dynamically based on user interactions and fetched data.

Styled Components: Styled components are utilized to create reusable and styled UI components. The code uses the styled function from the @emotion/styled library to define component styles using CSS-in-JS.

Theme Switching and Conditional Rendering: The app supports both light and dark themes. The ThemeProvider from the @emotion/react library allows dynamic theming based on user preferences. The app conditionally renders components based on the current page or moment (day/night) to display appropriate weather information and icons.

Functional Component Composition: The components are organized into functional components that receive props and render UI based on those props, enhancing code modularity and reusability.

Asynchronous Operations: The app handles asynchronous operations like data fetching using promises and async/await to ensure smooth user experience and accurate data display.
